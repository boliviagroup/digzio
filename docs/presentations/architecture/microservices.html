Backend: 8 Node.js Microservices

Independent Express.js services running on AWS ECS Fargate, communicating via REST and sharing a PostgreSQL database.
Auth Service
Handles user registration, login, JWT generation, and role-based access control (RBAC) for all users.
Port: 3001
Property API
Manages property listings, geospatial data (PostGIS), provider portfolios, and search filtering.
Port: 3002
KYC Service
Processes student verification, identity checks, and NSFAS status validation workflows.
Port: 3005
Institution API
Manages university data, student linking, enrollment verification, and nearby property queries.
Port: 3008
Application Service
Handles the tenant application lifecycle, from student submission to provider approval/rejection.
Port: 3006
Lease Service
Generates PDF lease agreements, manages digital signatures, and tracks active lease statuses.
Port: 3007
User Service
Manages extended user profiles, preferences, and cross-service user data synchronization.
Port: 3003
Notification Service
Integrates with AWS SES to send transactional emails, alerts, and system notifications.
Port: 3004
