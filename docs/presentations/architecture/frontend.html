A single-page application (SPA) providing distinct, tailored flows for Students, Providers, and Institutions, styled with utility-first Tailwind CSS.
The Mixed-Content Challenge
The frontend is served over HTTPS, but the AWS ALB endpoint is HTTP. Browsers block these "mixed content" requests, preventing direct API calls from the client.
The Reverse Proxy Solution
A Node.js/Express server serves the static React files and acts as a reverse proxy. It intercepts all calls to /api/* and forwards them server-side to the ALB, bypassing browser restrictions and CORS issues.
