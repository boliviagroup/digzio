Frontend — Web Platform

React + TypeScript + Vite: Modern, responsive web app

Core Framework React 18 + TypeScript, built with Vite for fast HMR and optimized bundles.
Styling & UI TailwindCSS utilities with Radix UI primitives for accessibility.
State Management React Query for server state and Zustand for light local state.
Forms & Validation React Hook Form paired with Zod schemas for robust validation.
Routing & Delivery Wouter for lightweight routing; served via Node/Docker on AWS ECS.
