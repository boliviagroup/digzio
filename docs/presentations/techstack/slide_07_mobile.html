Mobile App — React Native

One Codebase, Two Platforms — Built with Expo & React Native

Core Framework Expo SDK + React Native with TypeScript for a unified, type-safe codebase across iOS and Android.
Navigation & Styling Expo Router for file-based routing and NativeWind for TailwindCSS styling in React Native.
State & Forms TanStack Query for server state, Zustand for local state, and React Hook Form + Zod for validation.
Native Capabilities expo-camera and expo-image-picker for KYC document uploads, expo-secure-store for JWT storage.
Performance Shopify FlashList for high-performance lists, Reanimated + Gesture Handler for smooth interactions.
