Security Architecture

Security-First Design Across Every Layer

Authentication
JWT & Redis
Short-lived JSON Web Tokens with Redis-backed session management for immediate revocation capabilities.
bcrypt Hashing
Industry-standard bcrypt password hashing implemented securely within the isolated auth-service.
Secure Token Storage
Mobile app utilizes expo-secure-store (Keychain/Keystore) instead of vulnerable AsyncStorage.
Network & Infra
HTTPS & ALB
Strict HTTPS enforcement with SSL termination at the AWS Application Load Balancer level.
Rate Limiting
Express-rate-limit applied to auth, property, and notification services to prevent brute-force and DDoS attacks.
Strict CORS
Cross-Origin Resource Sharing configured per-service to only allow requests from verified Digzio domains.
Data & Application
Helmet.js
HTTP security headers automatically set across all 8 Node.js microservices to prevent common web vulnerabilities.
KYC Verification
Strict document verification workflow ensuring identity and proof of registration before platform access.
S3 Presigned URLs
Private S3 buckets for KYC and lease PDFs, accessed only via temporary, time-limited presigned URLs.
