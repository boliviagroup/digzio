Architecture Overview

A Production-Grade Microservices Platform Running on AWS Cape Town

9 Independent Services deployed on AWS ECS (Fargate) in af-south-1 (Cape Town) for minimal latency.
Containerized Infrastructure: Each service runs in Docker (Node.js 20 Alpine), ensuring consistent environments.
Traffic Management: All services sit behind an Application Load Balancer (ALB) with enforced HTTPS.
Data & Caching: Persistent data stored in TiDB, with Redis handling caching and sessions.
Content Delivery: Static assets and images served globally via AWS CloudFront CDN.
