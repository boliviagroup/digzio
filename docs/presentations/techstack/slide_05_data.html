Data & Storage

TiDB + Redis + AWS S3 + CloudFront — Built for Scale

TiDB
Primary Database
Distributed, MySQL-compatible database providing horizontal scaling and strict ACID transactions for core platform data.
Redis
In-Memory Cache
High-performance caching layer handling JWT session management, API rate limiting, and rapid API response caching.
AWS S3
Object Storage
Secure, scalable storage for all unstructured data including property images, KYC documents, and generated lease PDFs.
AWS CloudFront
Content Delivery Network
Global CDN ensuring low-latency image delivery and static asset distribution across all South African provinces.
AWS SES
Email Service
Reliable delivery of transactional emails for student applications, KYC status updates, and lease notifications.
