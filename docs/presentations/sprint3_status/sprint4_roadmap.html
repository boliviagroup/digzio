The Launch Strategy

Agreed path from Backend Completion to Full Platform
01
Connect the Website
Frontend team updates digzio.co.za to use the new AWS ALB endpoint and live APIs.
02
Soft Launch Testing
Run through the full student and provider workflows on the live website to validate the UX.
03
Sprint 4: Payments
Once the core rental loop is proven, integrate PayFast for secure rent collection and deposits.
04
Mobile App Development
Build the React Native iOS/Android app using the exact same, fully-tested backend APIs.
