AWS Infrastructure Overview

Production environment fully provisioned via Terraform
Compute & Network
ECS Fargate Cluster
Serverless container compute hosting all 8 microservices. Auto-scales based on traffic.
Application Load Balancer
Single entry point routing traffic to specific microservices based on URL paths.
Custom VPC
Isolated network spanning 3 Availability Zones with public/private subnet separation.
Data & Storage
RDS PostgreSQL 15
Primary relational database with PostGIS enabled for spatial property searches. Private subnet only.
ElastiCache Redis 7
In-memory caching layer to speed up frequent property queries and manage session state.
S3 Storage Buckets
Dedicated buckets for property images, KYC documents, and generated lease PDFs.
Delivery & Comms
CloudFront CDN
Global edge network delivering property images and static assets with millisecond latency.
AWS SES
Verified email sending service for transactional notifications (OTP, lease signatures, approvals).
Elastic Container Registry
8 private Docker image repositories storing the compiled microservice code.
