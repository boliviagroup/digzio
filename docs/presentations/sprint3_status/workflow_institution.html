Institution Workflow

Verify Students, Accredit Properties, Monitor Occupancy
/students
Student Verification
  • Review student registration requests linked to the institution.
  • Verify active enrollment status against university records.
  • Unlock specific university-only property listings for verified students.
/properties
Property Accreditation
  • Review provider properties submitted for university accreditation.
  • Apply "NSFAS Approved" or "University Accredited" badges.
  • Manage the list of approved housing providers.
/occupancy
Occupancy Monitoring
  • Real-time dashboard showing where verified students are placed.
  • Track total capacity vs. current occupancy in accredited buildings.
  • Export reports for university housing administration.
