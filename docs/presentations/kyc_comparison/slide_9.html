Integration Architecture

The 7-Step Asynchronous API Flow

1
Data Entry
Student submits SA ID number and name.
2
DHA API Request
Digzio Backend queries the DHA population register.
3
DHA Response
Returns valid, alive, and name match confirmation.
4
Smile SDK Capture
Browser launches SDK for a 3-second liveness selfie.
5
Liveness Job
Digzio Backend submits the liveness job to Smile Identity.
6
Webhook Callback
Smile Identity asynchronously returns the liveness result.
7
DB & WebSocket Push
Database updated to VERIFIED; real-time push to student app.
