POPIA Compliance Advantage

The Two-Layer Stack Dramatically Reduces Digzio's POPIA Liability by Storing Zero ID Images

The Protection of Personal Information Act (POPIA) requires organisations to minimise the personal information they collect and store.

The Two-Layer Stack eliminates this risk entirely. Because no ID card image is ever captured or transmitted, Digzio never holds this sensitive data. Only the DHA check result (valid/invalid) and the Smile Identity liveness result (passed/failed) are stored.
